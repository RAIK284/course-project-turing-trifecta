# WAVELENGTH

A web-based multiplayer game that mimicks the real life board game Wavelength.

### Tech Stack

Frontend:
- Typescript
- React
- MobX

Backend:
- C#
- .NET
- Entity Framework
- SignalR

### How to Run

See `wavelength-api/README.md` for backend instructions, and `wavelength/README.md` for frontend instructions.
